# AgroShield

AgroShield is a cybersecurity prototype designed to protect farmers from fraudulent agro suppliers, counterfeit agricultural inputs, and digital payment scams.

## Problem

Many farmers lose money due to:

- Fake fertilizers
- Fraudulent agro suppliers
- Scam messages requesting payments

These issues reduce agricultural productivity and financial security for farmers.

## Solution

AgroShield provides a digital verification platform that allows farmers to:

- Verify agro suppliers
- Check product authenticity
- Report suspicious phone numbers![{001C293E-E880-4748-8B92-576FC8A849D5} png](https://github.com/user-attachments/assets/2a1b1c68-ae4c-4115-a913-6d7ac03fb8eb)![{FFCEC5DA-A95E-4589-BD14-0DABB65B3762} png](https://github.com/user-attachments/assets/a63cf6c1-d234-4b76-8311-de068d07e3e7)
